A hub represents a node to which several endpoints, as well as other hubs can be connected. The FoxBox is the root hub. Simple devices that can do a single thing (e.g. a button) are endpoints, while complex devices containing several sensors or effectors are also hubs, in which each sensor and each effector is an endpoint.
